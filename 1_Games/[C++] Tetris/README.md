# Tetris
## Керування:
 * В грі:
 	* (← →) - рух вліво вправо
 	* ↓ - прискорити падіння
	* Space - поворот
	* Кнопки натистакються мишкою
 * В меню:
	* (↑ ↓) - виділти пункт
	* Space/Enter - вибір
 * ESC - назад

## Фичи:
 * Зчитування фігур з XML
 * Оригінальний стиль

## Зроблено на
 * C++
 * HGE16
 * tinyXml 1.0.1
 * Частини мого плеєра на основі Audiere 1.9.4
