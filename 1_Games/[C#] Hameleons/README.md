# Hameleons
 Гра про роботу з потоками
## Керування:
* ↑, ↓ - Вибрати пункт меню
* ←, → - В пункті _View cars_ і _Log_ прогортати таблицю
* Tab - перемикати активний елемент
* Enter - вибрати активний елемент

## Фичи:
 * Гарний ASCII вивід
