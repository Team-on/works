# Simulator of mass service systems (pizzeria)
## Controls:
* ↑, ↓ - Choose menu item
* ←, → - In _Доставка_ і _Склад_ scroll table

## Features:
 * Random building generator
 * Random interior generator
 * Random names generator
 * Cute ASCII output
 * Detailed info for delivery/stock/kitchen/client

## Made with:
 * C#
