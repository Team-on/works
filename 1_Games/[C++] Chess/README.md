# Chess
## Controls:
 * Mouse - вибір фігури. Вибрана фігура стає фіолетовою і потім можна вирішити куди їй ходити.
 * type OOr - short castling
 * type OOOr - long castling

## Features:
 * Mouse control in console
 * Cute ASCII output

## Made with:
 * C++
 * Underwater engine
 * My .ini parser

## Cheats:
 * suicide - lose game
 * mat - checkmate to white
 * chag - check to whute
